# 123 Game — Color Palette

Playful, high-contrast primaries for a children's counting game. Warm sunshine background with red/blue/green number bubbles.

| Token         | HEX       | HSL                  | Swatch |
|---------------|-----------|----------------------|--------|
| Primary       | `#FFD23F` | `hsl(46, 100%, 62%)` | 🟡 Sunshine Yellow |
| PrimaryGlow   | `#FF7A1A` | `hsl(23, 100%, 55%)` | 🟠 Tangerine |
| Accent        | `#FF3B6B` | `hsl(347, 100%, 62%)` | 🔴 Berry Red |
| AccentBlue    | `#3DB2FF` | `hsl(204, 100%, 62%)` | 🔵 Sky Blue |
| AccentGreen   | `#3CCB7F` | `hsl(146, 56%, 52%)` | 🟢 Mint Green |
| Background    | `#FFF8E7` | `hsl(46, 100%, 95%)` | ⬜ Cream |
| Foreground    | `#1F2937` | `hsl(217, 19%, 27%)` | ⬛ Slate Ink |

## Usage
- **Primary / PrimaryGlow** — main brand gradient (icon background).
- **Accent / AccentBlue / AccentGreen** — the three counting bubbles (1, 2, 3).
- **Background** — numerals & light surfaces.
- **Foreground** — wordmark on light, body text.
