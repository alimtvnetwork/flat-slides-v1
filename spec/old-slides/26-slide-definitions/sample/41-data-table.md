# 41 — Data Table (sample)

**Type:** `DataTableSlide`
**Seq:** 41
**Narrow idea:** Three plans, three price points.

## Why it is narrow
One axis of comparison (price by tier). Feature matrices, quotas, and add-ons each get their own slide.

## Capsules
Row-level accent only — `cream` / `gold` / `ember` left-cell bar. No standalone capsules.

## Animation
- Transition: `SlideIn` (rows arrive horizontally with the slide).
- Text animation: `Stagger` (header at 0.25s, rows at 35ms intervals).
- Reduced motion: instant — all rows visible, no stagger.

None.
